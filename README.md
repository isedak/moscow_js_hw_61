# Небольшое приложение с навигацией по страницам (Moscow)

**Задание:**

Создать приложение на произвольную тему с переходом по страницам с использованием обработчиков события кликов на ссылки/кнопки.

* Должны быть минимум 3 страницы.

* Можете использовать NavLink для создания ссылок

* Сделайте все красиво, можете использовать чистый CSS, либо готовые компоненты из различных UI-библиотек.
